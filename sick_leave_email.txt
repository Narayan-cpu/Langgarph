# Tools

You may call one function to assist with the user query.

You are provided with function signatures within <tools></tools> XML tags:
<tools>
{"name":"update","description":"Updates the document with the provided content.","parameters":{"properties":{"content":{"type":"string"}},"required":["content"],"type":"object"}}

{"name":"save","description":"Save the current document to a text file and finish the process.\n\n    Args:\n        filename: Name for the text file.","parameters":{"properties":{"filename":{"type":"string"}},"required":["filename"],"type":"object"}}
</tools>

# Getting Started

1. Review the available tools and their functions.
2. Determine which tool best suits your current task.
3. Use the appropriate function call to achieve your goal.
